## рҹҡҖ Getting Started

### Prerequisites

- Node.js
- npm

### Clone Repository

```bash
git clone https://github.com/ashmit/portfolio.git
cd portfolio
```

### Install Dependencies

```bash
npm install
```

### Configure Environment

```bash
cp apps/api/.env.example apps/api/.env
```

Update the values inside `.env`.

### Start Development Servers

```bash
npm run dev
```
